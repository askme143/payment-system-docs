사용 API

GET /admin/subscriptions

구독 목록, 상태, 다음 결제일, 결제 실패 여부를 검색합니다.

POST /admin/subscriptions/{subscriptionId}/adjust

관리자가 구독 상태, 다음 결제일, 실패 상태를 수동 보정합니다.

연관 잠재 위험
  • provider-paid-internal-unpaid PG 승인 성공 후 내부 미반영
  • idempotency-store-missing 멱등키 저장소 부재

상태 요약

시점구독 상태결제 상태설명
보정 전past_dueinternal unpaid / provider DONE토스 결제는 성공했지만 내부 상태가 결제 성공을 반영하지 못했습니다.
보정 후activepaid외부 결제 사실을 근거로 인보이스와 구독 상태가 일치합니다.
연기 전activepaid다음 결제일이 기존 주기에 맞춰 설정되어 있습니다.
7일 연기 후activepaid서비스 보상 기간만큼 다음 결제일이 뒤로 밀렸고 구독은 계속 이용 가능합니다.
결제 실패 해제 전past_duefailed자동 재시도는 끝났지만 고객이 별도 결제를 완료했거나 CS 예외 승인이 필요합니다.
정책 보정 후active 또는 canceledpaid 또는 adjusted운영 정책에 따른 최종 상태와 감사 로그가 함께 저장됩니다.

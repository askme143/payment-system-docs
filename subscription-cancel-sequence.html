구독 해지 플로우

회원이 기간 종료 시 해지 예약을 요청하면 현재 이용 기간은 유지하고, 기간 종료 이후 만료 배치가 구독을 canceled로 최종 마감하는 흐름입니다.

전체 시퀀스 목록전체 API 목록API 상세 설명MongoDB 구조 문서

사용 API

POST /subscriptions/{subscriptionId}/cancel

현재 이용 기간 종료 시 구독 해지를 예약하고 다음 정기 결제 대상에서 제외합니다.

POST /subscriptions/{subscriptionId}/resume

해지 예약 또는 결제 실패 상태의 구독을 재개합니다.

GET /subscriptions/me

현재 회원의 상품별 구독 목록, 이용 기간, 다음 결제일, 공통 기본 결제수단 표시 정보를 조회합니다.

POST /internal/subscription-billing/run

다음 결제일이 도래한 구독을 찾아 정기 자동결제를 실행합니다.

상태 요약

시점구독 상태결제 상태설명
해지 예약 전activepaid회원은 이용 중이고 다음 결제일이 예약되어 있습니다.
해지 예약 직후cancel_scheduledpaid현재 기간 권한은 유지되며 다음 결제일은 제거됩니다.
기간 종료 전cancel_scheduledpaid회원은 currentPeriodEnd까지 접근할 수 있고 해지 예약 철회가 가능합니다.
만료 배치 처리 후canceledpaid구독이 최종 종료되고 접근 권한이 회수되며 정기 과금 대상에 다시 포함되지 않습니다.
이후 구독 조회canceledpaid회원 화면은 재개 버튼 대신 신규 구독 또는 재구독 안내를 표시합니다.
